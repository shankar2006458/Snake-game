# 🐍 Snake Game

🔗 **Live Demo:** https://shankar2006458.github.io/Snake-game/

A modern, mobile-first implementation of the classic Snake game, built using **HTML Canvas, Tailwind CSS, and Vanilla JavaScript**. Designed for smooth gameplay, responsive controls, and a clean, premium UI experience.

---

## 🎮 About the Game

Snake is a classic arcade-style game where the player controls a growing snake, collecting food while avoiding collisions with walls and itself. The simplicity and replayability have made it one of the most iconic games ever created.

This version enhances that classic experience with modern UI, animations, and optimized performance.

---

## ✨ Features

* 🎯 **Classic Snake Mechanics**
* 📱 **Mobile-first gameplay** with D-pad controls
* ⌨️ **Keyboard controls** for desktop users
* 🔊 **Custom sound effects (no delay)**
* ⏸️ **Pause / Resume system**
* 🏆 **High score tracking (localStorage)**
* ⚡ **Dynamic difficulty (speed increases)**
* 🎨 **Modern UI with animations & glow effects**
* 📊 **Live score updates & feedback animations**

## 🧠 Tech Stack
